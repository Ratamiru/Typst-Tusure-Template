# Шаблон лабораторных работ для ТУСУР (Typst)

Шаблон для оформления лабораторных и практических работ по стандарту ТУСУР, написанный на [Typst](https://typst.app/) — современной альтернативе LaTeX.

Кому надоело бороться с LibreOffice, где титульник ездит сам по себе, а отступы живут своей жизнью — этот репозиторий для вас.

## Зачем Typst, а не Word/LibreOffice/LaTeX?

| Инструмент | Проблема |
|---|---|
| Word / LibreOffice | Постоянно ломается вёрстка, особенно титульный лист |
| LaTeX | Мощный, но перегружен для простых отчётов |
| Markdown | Слишком простой, нет нормального контроля оформления |
| **Typst** | Простой синтаксис + точный контроль над вёрсткой ✓ |

## Что внутри

```
├── title.typ        # Титульный лист по стандарту ТУСУР
├── main.typ         # Основной документ (сюда пишете содержание работы)
├── main.pdf         # Пример скомпилированного PDF
├── tusur-lab.skill  # Скилл для Claude Code (см. ниже)
└── 1.png, 2.png, 3.png  # Скриншоты из примера лабораторной
```

### title.typ
Готовый титульный лист: шапка министерства, название университета и кафедры, поля для студента и преподавателя. Нужно только поменять имя, группу и дисциплину.

### main.typ
Основной документ. Подключает титульник и содержит пример оформленной лабораторной работы (Android-калькулятор).

## Быстрый старт

**Установить Typst:**
```bash
# Linux (через cargo)
cargo install typst-cli

# или скачать бинарник с https://github.com/typst/typst/releases
```

**Скомпилировать в PDF:**
```bash
typst compile main.typ
```

**Отредактировать под себя:**
1. В `title.typ` замените группу, ФИО студента и преподавателя, дисциплину
2. В `main.typ` напишите содержание вашей лабораторной

## Интеграция с Claude Code (опционально)

В репозитории есть скилл `tusur-lab.skill` для [Claude Code](https://claude.ai/code). Он позволяет скормить PDF с заданием на лабу и получить готовый Typst-документ.

Достаточно написать: *«напиши лабу по этому заданию»* и приложить PDF — Claude сгенерирует структуру отчёта по шаблону ТУСУР.

## Зависимости

- [Typst](https://typst.app/) — компилятор документов
- Шрифт **Times New Roman** (используется в шаблоне по требованиям ТУСУР)
